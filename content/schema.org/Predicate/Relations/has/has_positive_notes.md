Provides positive considerations regarding something, for example product highlights or (alongside [negativeNotes](negativeNotes)) pro/con lists for reviews.&lt;br/&gt;&lt;br/&gt;

In the case of a [review](schema.org/Type/is_a_/creative_work/review.md), the property describes the [itemReviewed](itemReviewed) from the perspective of the review; in the case of a [product](schema.org/Type/is_a_/product.md), the product itself is being described.&lt;br/&gt;&lt;br/&gt;

The property values can be expressed either as unstructured text (repeated as necessary), or if ordered, as a list (in which case the most positive is at the beginning of the list).

Relation describes that: 
[ #has_/domain  :: Product, Review ]
( #has_/name :: is_positive_notes )
( #has_/range :: ItemList, ListItem, Text, WebContent )
